# v2-frontend

Frontend for toystori using [reasonml](https://reasonml.github.io/)

## Local setup

**Dependencies**

```bash
npm install -g yarn
yarn install
```

**Everyday tasks**

```bash
  yarn start
    Starts the development server.

  yarn build
    Bundles the app into static files for production.

  yarn test
    Starts the test runner.
```
